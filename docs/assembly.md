## Basic bring-up checklist

1. Verify supply voltages before inserting the ESP32-C3.
2. Verify I2C communication with the DS3231.
3. Verify endstop polarity.
4. Test LED and button.
5. Test motor actuation with no food loaded.
6. Confirm one full mechanism cycle is detected correctly.
7. Set RTC time.
8. Test one automatic schedule.
