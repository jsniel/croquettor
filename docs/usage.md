# Usage

## Manual feeding

- Short press the local button.
- Or connect to the AP and use the web button.

## Configuration portal

- Hold the button for about 3 seconds.
- Join the `croquettor` Wi-Fi AP.
- Open `http://192.168.4.1/`.

## Automatic schedules

Two schedules are available.

Each schedule includes:

- enable flag,
- hour,
- minute,
- number of doses.

## Important operational note

Current firmware checks schedules every 10 seconds. In practice this is enough for minute-level scheduling, but precise behavior depends on RTC validity and loop health.
